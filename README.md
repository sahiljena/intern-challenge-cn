# intern-challenge-cn

## Installation
### Clone this git repository
``` 
git clone https://github.com/sahiljena/intern-challenge-cn
```
### Cd into working folder
```
cd intern-challenge-cn 
```
### Install depedencies
``` 
npm install
```

### Start the React App
```
npm start
```
### Start the Json Server
```
npm run dev
```
